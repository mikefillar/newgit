Configuring Git Repository

  1. go to vscode and click terminal arrow then terminal
  2. Check if git is already installed by typing git --version in your command line
  3. Configure git in your local computer
    • Set your username
      • git config --global user.name "userdefinedname"
    • Set your email address
      • git config --global user.email "userdefinedemail"
  4. To initialize the project directory type git init

Git add and commit

  1. to see the status of the file in your directory git status
  2. to track new file use the command git add
    Syntax git add . for group files
    Syntax git add filename for individual file
  3. by running git status again you can see staged files in screen ready for commit
  4. Commit the file you staged in your local repository by typinggit commit -m "sample message"
